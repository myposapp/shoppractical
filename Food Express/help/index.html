B) Directories & File Uses

Our Design uses only assets & lib folder. So, don't touch any other folder for ui changes.

    • assets(Folder) - Put all Static images & other data (in Our Case images & json file) in this folder
    • fonts(Folder) - Put all font file in this folder
    • lib(Folder) - Main folder that contains all the dart code
      • appBehaviour(Folder) - App behaviour file for our app. (Remove ripple effect).
      • constant(Folder) - Contain Constants (fixPadding, textstyle, etc) in this folder
      • Pages(Folder) - Contain All App Pages in this folder except main.dart
        • confirm_order_add_address(Folder) - Contain Confirm order page and add new address page code in this folder.
        • favourite(Folder) - Contain all favourite tab related pages in this folder.
        • home(Folder) - Contain All the Component that use in Discover Page in this folder
        • login_signup(Folder) - Contain Login, register and otp Page
        • nearby(Folder) - Contain Nearby tab Page
        • onboarding(Folder) - Contain all Onboarding relared file
        • order(Folder) - Contain all order tab relared file
        • profile(Folder) - Contain All Profile related files in this folder
        • restaurant(Folder) - Contain Restaurant Page related file in this folder
        • search(Folder) - Contain Search Page related file in this folder
        • notifications.dart(File) - Notification Page
        • splashScreen.dart(File) - Splashscreen code
    • widget(Folder) - Contain all our own created widget file. (Column Builder).
    • main.dart - Entry Point of Our App.
  • pubspec.yaml(File) - Specification file (All the package and local assets (Like Images, Fonts, etc.) we use in this application is specify in this file.).
  • Packages (Dependencies)

C) About Google Map Configuration

We use google_maps_flutter package for google map.

You need Google Map Api key for Accessing Google Map.

  • If you want to know how to get Google Map Api Key then please visit this link google_maps_flutter.

    • 1) Android

  • Put that api key in 2 places in our code.

    2) iOS

  • Put that api key in 2 places in our code.
